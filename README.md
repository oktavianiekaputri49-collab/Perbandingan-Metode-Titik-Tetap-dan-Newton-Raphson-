# Perbandingan-Metode-Titik-Tetap-dan-Newton-Raphson-
Mini project metode numerik membandingkan Metode Iterasi Titik Tetap dan Metode Newton-Raphson untuk menyelesaikan persamaan nonlinear f(x)=−(x^2 +1.8x+2.5).  Dalam project ini dilakukan analisis proses iterasi kedua metode untuk melihat kecepatan konvergensi menuju solusi dengan bantuan Microsoft Excel.
